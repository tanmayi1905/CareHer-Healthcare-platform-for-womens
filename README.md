# CareHer 🌸

**CareHer** is a dedicated healthcare platform designed to support women at every stage of life. It combines health-tracking tools, personalized wellness guidance, and a supportive community space — all in one place.

🔗 Live site: [care-her.netlify.app](https://care-her.netlify.app)

## About

CareHer aims to create a safe, supportive, and empowering space for women's physical and emotional well-being. The platform focuses on early detection, self-care tools, and personalized recommendations to help users take charge of their health.

## Features

- **Health Check** — Tools for early detection of PCOD and PCOS
- **BMI Calculator** — Smart calculator to track body mass index
- **Period Tracker** — Intelligent tracker to monitor menstrual cycles accurately
- **Personalized Diet Plans** — Diet recommendations based on individual health inputs
- **Product Recommendations** — Customized wellness product suggestions
- **Exercise** — Guided exercise routines for physical well-being
- **Reviews** — Honest user reviews and feedback
- **Contact** — Easy access to get in touch with the team
- **Sign Up / Login** — User account creation and access
- **Dark Mode** — Toggle for light/dark theme (🌙)

## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- A modern web browser
- (Optional) [Node.js](https://nodejs.org/) if the project uses a build tool

### Installation

```bash
# Clone the repository
git clone https://github.com/tanmayi1905/careher.git

# Navigate into the project directory
cd careher

# Open index.html in your browser, or serve it locally
```

### Running Locally

If this is a static site, you can simply open `index.html` in your browser, or use a local server:

```bash
# Using VS Code Live Server, or:
npx serve .
```

## Deployment

This project is deployed on **Netlify**. Any push to the main branch (if connected via Netlify's CI) will trigger an automatic redeploy.

To deploy manually:

```bash
netlify deploy --prod
```
